# X-SyncTrace v4

**Advanced Forensic Browser Artifact Analyzer + UBPA**

A powerful, easy-to-use digital forensics tool for analyzing Chrome/Edge browser history, sync artifacts, and user behavior patterns.

<img width="1024" height="1024" alt="logo" src="https://github.com/user-attachments/assets/f4b65a5c-7f55-4580-ae85-e17bc3511b36" />




### ✨ Key Features

- Full browser history analysis
- Sync events & device tracking
- **UBPA** — User Behaviour Pattern Analysis Engine
- Anomaly detection (late night, bursts, suspicious domains)
- Forensic report with hash verification
- Admissibility checklist (Court-ready)
- Python collector script included

### 📥 How to Use

1. Download the [Python Collector](xsynctrace_collector.py) and run it on the target machine
2. Or use the built-in TXT/CSV converter
3. Drop the generated `xsynctrace_export.json` into the tool
4. Analyze → Generate Report → Export for Court

### 📁 Files Included

- `index.html` → Main Tool (Single File)
- `xsynctrace_collector.py` → Forensic Collector Script
